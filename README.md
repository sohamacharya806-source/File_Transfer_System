# File_Transfer_System
This project is a file transfer system that sends pdf,txt,jpg,etc files to different computers which are connected to the server and whose server address is known

How to start the file transfer system?
First step is to start the server with the code
.\server.exe

next step is to start the client.c in file in the laptop which is going to share its file the code is
.\client.exe <enter ip of the computer to which we sent the file> <file name which is to be sent>

you can use ip = 127.0.0.1  to send the file to another terminal in your computer


next step is to shut the server off by using the exit command
